Principais formas de formatação

Negrito/Destaque

Nesta frase temos um termo em NEGRITO usando a TAG "b", que não é uma TAG semântica.

Nesta frase temos um temor em NEGRITO usando a TAG "strong", que é uma TAG semântica.

Itálico/Ênfase

Nesta frase temos um termo em Itálico, usando a TAG "i".

Nesta frase temos um termo em Itálico, usando a TAG "em".

Texto Marcado

Podemo criar também um texto marcado usando a TAG "mark"

E no outro parágrafo, temos outro texto marcado com a mesma TAG.

Texto GRANDE e texto pequeno

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o sublinhado, porém usamos uma TAG sem semântica, a TAG "u".
